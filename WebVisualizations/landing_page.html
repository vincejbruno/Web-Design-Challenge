Project Description

The purpose of this project was to examine various changes in weather patterns as one moves closer to the equator. The Open Weather Map API was used to pull data for over 500 cities. The cities used in this analysis were randomly selected using the API, and all data is related to 01/05/2017.

The data was cleaned, and MatPlotLib was utilized to build the following charts:
